#### 使用

```shell
# clone 脚手架
git clone https://github.com/LeiHanCN/react-cli.git <project_name>
cd <project_name>/scripts
chmod 777 init.sh
# 运行初始化脚本
./init.sh
```

#### 常用指令

```shell
# 开发
yarn dev
# 打包
yarn build
# 提交代码
yarn cz
```
